This chapter will show you how to install the script onto your server.

When you upload the contents of upload.zip into the public directory of your server and visit your website, You should see the BitFlan Installer.

The Installer is a wizard that helps you setup the script onto your website.

Before you can continue with the installation process, You must make sure that your server matches these requirements:

  • PHP 8.x
  • MySQL Database
  • Storage & Public Directories writable.
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO MySQL PHP Extension
  • Tokenizr PHP Extension
  • XML PHP Extension
  • cURL PHP Extension
  • ZIP PHP Optional

Step 1 - License Verification

Before you can install the script onto your website, you must verify your license. To verify your license, you need your envato purchase code. Click Here if you donâ€™t know where to find your purchase code.

Note: If you are installing on a local machine or If you have an extended license, You do NOT have to link your domain name. Simply type in your purchase code while installing and It will work

After you find your purchase code, You must link your purchase code to a Domain Name by Clicking Here. Make sure to enter your Domain Name without the HTTPS / WWW or any protocol (i.e mywebsite.com)

After attaching the domain to your purchase code, Go back to the installer, paste your purchase code and click on the Verify button to continue. If all is done properly, you should now move onto the database section.

Step 2 - Database Setup

You need a MySQL database for use with this project. We recommend creating a new Database. After creating the database, you just need to fill out the respective details in the installer.

  • Database Name in the Name field.
  • Database Port in the Port field.
  • Database Host in the Host field. (Usually localhost)
  • Database Username in the Username field.
  • Database Password in the Password field.

Step 3 - Admin Setup

After setting up the database, you must create your admin user. Fill out the respective details and make sure your websiteâ€™s URL is valid. Click on the Button to complete the installation process.

If all goes well, you will be redirected to the Admin Panel. Login with your newly created account and enjoy the product.
