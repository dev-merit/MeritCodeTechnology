This Page includes some Important Notes / Known Issues and their solutions with the script

500 Error when visiting your Website

If you see a 500 Server Error when you visit your website for the first time (after uploading contents of upload.zip to your public directory), It probably means that the OpenSSL extension is not enabled on your server.

Make sure to enable the OpenSSL extension as-well as check that all other requirements match (as described in Installation)

If you still have issues, Please contact us at Our Support Desk

If you try to edit the Tool Titles & Descriptions in the Web Tools Settings section of the admin panel but you canâ€™t see them changing on the main website, It is because you have the option Translate Tool Titles & Descriptions in Language Settings turned on.

If Language Settings > Translate Tool Titles & Descriptions is turned on, then the titles and descriptions from the multi-lingual translation files will be used instead of the settings. The Multi-lingual translation files are inside the lang directory of the script files.

Check Translation for more information.
