This page provides information about individual tool settings.

Every tool has Title, Description settings, but certain tools have special settings unique only to them. This page will specify those tools and tell you how to configure them.

Domain Generator

Domain Generator has a unique setting for Affiliate Links. Affiliate links will help you earn a commission If a user clicks on the button and purchases a domain through your affiliate URL.

You can configure the Affiliate URL in Web Tools Settings > Domain Generator.

You can use the affiliate provider of your choice. All you have to do is replace the Domain Name in the affiliate link with the {name} wildcard.

For example: https://myaffiliate.com/purchase?domain=mydomain.com will become https://myaffiliate.com/purchase?domain={name}
